# LINE Command Center

ผู้บริหารควบคุมทุกอย่างผ่าน LINE โดยไม่ต้องเปิดคอมพิวเตอร์

---

## แนวคิดหลัก

> ทุกคนใช้ LINE อยู่แล้ว — ไม่มีช่วงการเรียนรู้ (Zero Learning Curve)

แทนที่ผู้บริหารจะต้องเข้า web dashboard เพื่ออนุมัติงานหรือดูรายงาน ระบบส่งทุกอย่างเข้า LINE โดยตรง

---

## สิ่งที่ทำได้ผ่าน LINE แล้ว (ปัจจุบัน)

| สิ่งที่ทำได้                        | วิธี                        |
| ----------------------------------- | --------------------------- |
| อนุมัติ/ปฏิเสธ expense              | กด ✅ ❌ บน Flex Message      |
| อนุมัติ/ปฏิเสธสมาชิกใหม่            | กด ✅ ❌ บน Flex Message      |
| รับแจ้งเตือนเมื่อ expense ถูกบันทึก | push notification อัตโนมัติ |

---

## สิ่งที่จะทำได้ใน Phase 2–5

| คำสั่ง                  | Phase | ผลลัพธ์                           |
| ----------------------- | ----- | --------------------------------- |
| `สรุปวันนี้`            | 2     | ยอดขาย + ค่าใช้จ่าย + กำไรสุทธิ   |
| `สรุปเดือนนี้`          | 2     | ภาพรวมการเงินรายเดือน             |
| `ดีลรออนุมัติ`          | 2     | รายการ deal/expense ที่รอ         |
| `สต๊อกวิกฤต`            | 2     | สินค้าใกล้หมดคลัง                 |
| `ลูกค้าใหม่วันนี้`      | 2     | รายชื่อลูกค้าใหม่                 |
| `ตัวชี้วัดทีมขาย`       | 2     | Win Rate, Pipeline, ยอดแต่ละเซลส์ |
| `✅ [รหัส]` / `❌ [รหัส]` | 2     | อนุมัติ/ปฏิเสธผ่านโค้ด            |
| ถามอะไรก็ได้            | 4     | AI Bot ตอบเอง                     |

---

## Architecture

```
ผู้บริหารพิมพ์ใน LINE
    → LINE Webhook → /api/line/webhook/main
    → parse คำสั่ง
    → ดึงข้อมูลจาก DB
    → ตอบกลับ Flex Message / Text
```

ใช้ infrastructure เดิม: [[ระบบ LINE]] + [[Infrastructure]]

---

## ขั้นตอนการพัฒนา

1. **ปัจจุบัน** — approval flow เท่านั้น
2. **Phase 2** — เพิ่ม text command parser + query endpoints
3. **Phase 4** — แทนที่ rule-based parser ด้วย AI Bot
